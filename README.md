# Fuck

## What the Fuck?

Fuck is a small animation code (inspired by sl) that works on your terminal and allows you to express your negative feelings about some situations.

you do not have to hate, you can love with fuck too. (with -l or --love flag)

## Installation 

Fuck works for only linux platforms

**Note:** before using fuck you should download ncurses library to your computer.

I am not planing to write a installation script for a while so you can just move the binary file to your /home/bin or wherever you want and if you want to inspect the source code it is already there!
